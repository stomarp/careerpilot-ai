# CareerCopilot AI

CareerCopilot AI is an AI-powered career assistant that helps job seekers analyze resumes, compare them with job descriptions, calculate ATS scores, detect missing skills, generate interview questions, track applications, and create personalized learning roadmaps.

## Features

- Resume upload and parsing
- Job description analysis
- ATS score calculation
- Skill gap analysis
- AI-generated interview questions
- Application tracker
- Personalized learning roadmap
- Future: mock interviews and voice mode

## Tech Stack

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: FastAPI, Python
- Database: PostgreSQL
- AI: OpenAI API, Embeddings
- Vector DB: ChromaDB
- Deployment: Docker
