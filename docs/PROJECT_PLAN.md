# CareerCopilot AI Project Plan

## Goal

CareerCopilot AI is an AI-powered career assistant that helps job seekers analyze resumes, compare them with job descriptions, calculate ATS scores, detect missing skills, generate interview questions, track applications, and create learning roadmaps.

## Phase 1: Core MVP

- Resume upload
- Job description upload
- Resume parsing
- JD parsing
- ATS score calculation
- Skill gap analysis
- AI-generated interview questions

## Phase 2: Career Management

- Application tracker
- Recruiter email drafting
- Company preparation

## Phase 3: AI Coaching

- Mock interviews
- Answer evaluation
- Learning roadmap
- Voice mode

## Tech Stack

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: FastAPI, Python
- Database: PostgreSQL
- AI: OpenAI API
- Vector DB: ChromaDB
- Infrastructure: Docker
